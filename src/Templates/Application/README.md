# Sisig Factory

Sisig Factory Sales & Inventory System

## Getting Started

``` bash
$ composer install
$ php vendor/bin/phinx migrate && php vendor/bin/phinx seed:run
$ php -S localhost:8000 -t public/
```

Then go to [http://localhost:8000](http://localhost:8000).
